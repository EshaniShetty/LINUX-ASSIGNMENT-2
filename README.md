# LINUX-ASSIGNMENT-2

Linux Programming Assignment 2: A practical guide to terminal commands, filesystem navigation (absolute vs. relative paths), file search utilities, and permission management using chmod, chown, and chgrp.


Objectives
This repository contains solutions for Linux Programming Assignment 2, focusing on terminal mastery and filesystem administration.
